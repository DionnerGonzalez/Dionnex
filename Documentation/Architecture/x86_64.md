# x86_64 Architecture Overview

## Long Mode (64-bit) Transition
- Kernel expects Multiboot2 or UEFI loader handover in 64-bit Long Mode.
- Page tables: 4-level PML4 paging (or 5-level PML5 if supported by CPU).
- CR0, CR3, CR4 and EFER MSRs configured for NX (No-Execute) and SMEP/SMAP bit protection.

## Interrupt Vector Mapping
- Vectors 0-31: CPU Architecture Exceptions (Page Fault #PF, Double Fault #DF, GPF #GP).
- Vectors 32-255: APIC I/O and Local Interrupts (APIC Timer, Spurious, IPIs).
- SYSCALL/SYSRET instructions configured via `IA32_LSTAR` MSR.
